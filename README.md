# 📊 Dashboard de E-commerce Olist

Dashboard de Business Intelligence desenvolvido em Python com dados reais do e-commerce brasileiro Olist. Analisa mais de 100 mil pedidos exibindo KPIs e gráficos interativos.

## 🚀 Funcionalidades

- KPIs de faturamento total, ticket médio, máximo e mínimo
- Gráfico de formas de pagamento
- Gráfico de faturamento por mês
- Gráfico de status dos pedidos

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Numpy
- Plotly
- Dash
- Dash Bootstrap Components
